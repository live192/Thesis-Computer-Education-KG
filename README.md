## 快速开始
1.  克隆本仓库：`git clone https://github.com/现场192/Thesis-Compuer-Education-KG.git`
2.  后端部署：`cd src/backend && pip install -r requirements.txt`
3.  前端运行：`cd src/frontend && npm install && npm run dev`
